Remapping the Caps lock key to Ctrl

May 18, 2022 · 1 min

Introduction#

I don’t find myself using the Caps lock key that much and would rather have it mapped to Control for convienience. While this is really straight-forward on a Mac, it requires a non-obvious solution on Windows. Luckily, SO came to the rescue.

The Recipe#

We need to run Powershell as an administrator and type the following commands:

$hexified = "00,00,00,00,00,00,00,00,02,00,00,00,1d,00,3a,00,00,00,00,00".Split(',') | % { "0x$_"};

$kbLayout = 'HKLM:\System\CurrentControlSet\Control\Keyboard Layout';

New-ItemProperty -Path $kbLayout -Name "Scancode Map" -PropertyType Binary -Value ([byte[]]$hexified);

After reboot, the caps lock key will be mapped to Ctrl
